# MFFTrack

PyTorch implementation of our MFFTrack tracker.

The results for four tracking benchmarks (UAV123, GOT10K, LaSOT) have been released. 

## 1. Dataset Introduction

This code is compatible with five challenging tracking benchmarks, i.e., UAV123, GOT10K, UAV123@10fps, UAVTrack112_L and DTB70.
